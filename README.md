
# 📦 core-ui-admin

這是 **pinpinAura** 專案專屬的後台管理儀表板，基於 [CoreUI Free Bootstrap Admin Template](https://github.com/coreui/coreui-free-bootstrap-admin-template) 開發並進行客製化調整。

## 📚 功能特色

- 基於 CoreUI Free (MIT 授權)
- 整合 React + TypeScript 技術
- 模組化資料夾結構，方便維護與擴充
- 已準備好 API 整合與主題切換功能
